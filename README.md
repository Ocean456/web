## 员工信息管理系统

### 需求分析

1. 注册登录 （分为 管理员和 普通用户）
2. 管理员：
    1. 登录 -lmx
    2. 员工信息（显示） -lmx
        1. 员工编号
        2. 员工姓名
        3. 员工性别
        4. 员工年龄
        5. 员工电话
        6. 员工地址
        7. 员工部门
        8. 员工职位
        9. 员工工资
        10. 员工入职时间
        11. 员工离职时间
        12. 员工紧急联系人电话
        13. 员工宿舍
    3. 员工入职与离职 -xsd
    4. 员工信息分析 -sdy
    5. 员工信息查询 -sdy
    6. 员工信息修改 -lmx
    7. 薪水调整 -xsd
    8. 员工宿舍调整 -sdy
3. 普通用户：
    1. 登录（员工编号，员工编号后六位） -lmx
    2. 个人信息完善 -xsd
